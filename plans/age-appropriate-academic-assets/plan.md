# 학년군 적합 학습 자산 Plan

## 목표

메인 전투의 학생과 학습 몬스터가 현재 나이/학년군에 맞게 보이도록 한다. 특히 초등학생과 중학생이 직장인처럼 보이는 문제를 줄이고, 학년이 올라갈수록 학습 압박과 시험 소재가 자연스럽게 강해지게 만든다.

## 분석

- `data/grade_visuals.json`은 초1~초6, 중1~중3, 고1~고3, 1수~N수까지 16개 성장 구간을 이미 가진다.
- `tools/build-visual-assets.mjs`가 학생 16프레임과 메인 몬스터 192프레임을 결정적으로 생성한다.
- 기존 학생은 체형과 복장이 비슷해 초등/중등/고등의 차이가 충분히 크지 않았다.
- 기존 몬스터는 학년군별 이름은 좋지만 실루엣 검수 정보가 부족해 컨택트시트에서 나이대 적합성을 확인하기 어려웠다.

## 제작 기준

- 초등 저학년: 작은 체형, 노란 안전모, 밝은 책가방, 크레파스/연필.
- 초등 고학년: 조금 큰 체형, 교과서, 발표 노트, 실험 관찰장.
- 중등: 교복 리본, 조끼, 수행평가 파일, 학생 가방.
- 고등: 어두운 교복, 학교 배지, OMR/모의고사 종이, 두꺼운 책가방.
- N수: 후드, 헤어밴드, 커피, 독서실 종이, 형광펜.
- 일반 몬스터는 학년군별 학습 물체로 만들고, 보스는 크기와 장식 밀도를 올린다.

## 작업 순서

1. 학생 스프라이트의 학년군별 실루엣과 소품을 분리한다.
2. 메인 몬스터 192프레임을 초등/중등/고등/N수 학습 소재로 재생성한다.
3. 컨택트시트에 나이, 학년군, 학생 타이틀, 몬스터명을 표시한다.
4. `visual:qa`와 `verify:mobile`로 아틀라스, 실제 화면, 모바일 레이아웃을 검증한다.

## 완료 기준

- 초등학생이 성인 직장인으로 보이지 않는다.
- 중고등학생의 복장이 회사 정장이 아니라 학교 교복/수험생 소품으로 읽힌다.
- 몬스터가 각 학년군의 학습 환경과 시험 단계에 맞게 보인다.
- 컨택트시트에서 사람이 학년군 부적합 자산을 즉시 찾아낼 수 있다.
