# 학생탭 배경 길감/접지감 폴리싱 계획

## 목표

학생탭 전투 배경이 캐릭터와 따로 노는 문제를 해결한다. 원정대 화면처럼 캐릭터와 몬스터가 같은 길 위에서 이동하고 전투하는 느낌을 주는 것이 목표다.

## 현재 문제

- `presentation.backdrop.panWidthPercent=1400` 때문에 화면에 배경 원본의 좁은 구간만 확대되어 보인다.
- 기존 학생 배경 원화는 퀄리티는 괜찮지만, 하단 바닥과 전투 레인의 경계가 약해 캐릭터 발이 배경에 붙지 않는다.
- 같은 학년/과정 안에서 배경이 길게 이어지는 느낌은 있으나, 원화 안의 이동로가 명확하지 않아 방치형 RPG식 진행감이 약하다.

## 작업 방향

1. 기존 원화급 `asset-004.png`를 유지하되, 전투용 파노라마 산출물에 학년군별 러닝 레인을 합성한다.
2. 배경 줌을 낮춰 교실/복도/독서실의 공간감이 더 많이 보이게 한다.
3. 러닝 레인의 상단 경계, 접지 그림자, 타일/나무결, 하이라이트를 추가해 캐릭터가 같은 바닥 위에 서 있는 느낌을 만든다.
4. 러닝 레인 관련 수치는 `data/battle_road_config.json`에서 조정할 수 있게 하고, 각 파라미터에 한글 도움말을 둔다.
5. `visual:smoke`, `live:polish`, `verify:mobile`로 실제 모바일 화면에서 품질을 확인한다.

## 검증 기준

- 학생탭 배경에 기존 data image 파노라마가 적용되어야 한다.
- 기존 CSS 배경/격자는 숨겨져야 한다.
- 배경 pan width는 과확대 값으로 돌아가지 않아야 한다.
- 전투 화면 캡처에서 캐릭터와 몬스터 발 위치가 하단 러닝 레인 안에 들어와야 한다.
- 가로 오버플로, 스프라이트 상하 클리핑, 콘솔 에러가 없어야 한다.
