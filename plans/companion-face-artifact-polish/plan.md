# 직업동료 얼굴 아티팩트 폴리싱 계획

## 배경

원정대/직업동료 확대 리뷰 시트에서 일부 여자 동료 얼굴 주변에 둥근 장식처럼 보이는 아티팩트가 확인됐다. 상용 기준에서는 얼굴 가독성과 누끼 안정성이 먼저 보장되어야 하므로, 원본 생성 규칙과 자동 검증을 함께 보강한다.

## 작업 항목

1. 문제가 보이는 직업동료 원본 PNG와 확대 리뷰 시트를 확인한다.
2. 얼굴 근처에 붙는 원형 액세서리, 큰 둥근 옆머리, 헤드셋 패드를 줄인다.
3. 직업 특성은 얼굴 장식이 아니라 의상, 모자, 소품, 색상으로 읽히게 한다.
4. `professional_sprite_manifest.json`에 생성 프로필을 기록하고 face-safe 검증을 추가한다.
5. 직업동료/원정대 몬스터 스프라이트를 재생성하고 리뷰 시트를 다시 만든다.
6. `asset:factory:review`, `asset:factory:qa`, `verify:mobile`, `live:polish`로 최종 검증한다.

## 완료 기준

- 얼굴 위 원형 장식이나 불명확한 덩어리가 확대 리뷰 시트에서 보이지 않는다.
- 직업동료 방향은 오른쪽, 원정대 몬스터 방향은 왼쪽을 유지한다.
- 누끼, 클리핑, 축, 프레임 키 흔들림 검사가 실패 없이 통과한다.
- face-safe 규칙을 위반하면 `npm run asset:integrity`가 실패한다.
